§ 2.6

Cylinders and Quadric Surfaces

Definition · Cylinder
A cylindrical surface (cylinder) is a surface generated by all lines (rulings) parallel to a given line and passing through a curve in a plane. A variable that is absent from the equation is the free variable — rulings are parallel to that variable's axis.
Definition · Trace
The trace of a surface is the cross-section obtained by intersecting the surface with a plane parallel to one of the coordinate planes (set one variable to a constant).
Surface Name Standard Equation Key Characteristic
Ellipsoidx²/a² + y²/b² + z²/c² = 1All traces are ellipses (or circles)
Elliptic Paraboloidz = x²/a² + y²/b²Opens along free axis; circular cross-sections
Hyperbolic Paraboloidz = y² − x²Saddle shape; xy-trace gives crossing lines
Elliptic Conez² = x²/a² + y²/b²Two nappes; vertex at origin
Hyperboloid of One Sheetx² + y² − z² = 1Connected; waist at xy-plane is a circle
Hyperboloid of Two Sheetsz² − x² − y² = 1Two separate pieces; no xy-trace (no real soln)
Cylinderx² + y² = r² (or similar)Missing variable = free variable; rulings ∥ that axis
⚠ Exam Tip
For each surface, be able to: (1) Name it, (2) Describe its traces/level curves, (3) If it's a cylinder, identify the free variable and the direction of the rulings. Know that \(z^2 = x^2 + y^2 + 10\) has no real solution when \(z^2 - x^2 - y^2 = 10\) and \(x=y=0\) gives \(z=\pm\sqrt{10}\), so it's a hyperboloid of two sheets; conversely \(z^2 = x^2 + y^2 - 10\) requires \(x^2+y^2 \ge 10\) — also two sheets.
§ 4.1

Functions of Several Variables

Definition
A function \(f(x,y)\) assigns exactly one value \(z\) in some set \(B\) to every ordered pair \((x,y)\) in its domain \(A \subseteq \mathbb{R}^2\).
Domain: All \((x,y)\) where \(f\) is well-defined.  |  Range: \(\{f(x,y)\mid (x,y)\in \text{Domain}\}\)
Level Curves
The level curves of \(f(x,y)\) are curves satisfying \(f(x,y) = k\) for constants \(k\) in the range of \(f\). They are subsets of the domain (in the \(xy\)-plane).
Example — Domain & Range
Let \(f(x,y) = \sqrt{1 - x^2 - y^2}\).
Need \(1 - x^2 - y^2 \ge 0 \Rightarrow x^2+y^2 \le 1\).
Domain: \(\{(x,y)\mid x^2+y^2 \le 1\}\) (closed disk of radius 1).
Range: \([0,1]\).
Example — Level Curves
Let \(f(x,y) = 5 - \sqrt{x^2+y^2}\). Level curves: \(5-\sqrt{x^2+y^2}=k \Rightarrow x^2+y^2 = (5-k)^2\).
These are concentric circles centered at the origin with radius \(5-k\).
§ 4.2

Limits and Continuity

Definition · Limit
\(\displaystyle\lim_{(x,y)\to(a,b)} f(x,y) = L\) iff for all \(\varepsilon > 0\), there exists \(\delta > 0\) such that \(|f(x,y)-L| < \varepsilon\) whenever \(0 < \sqrt{(x-a)^2+(y-b)^2} < \delta\).
Two-Path Test (Showing a Limit DNE)
If \(f(x,y)\) approaches two different values along two different paths to \((a,b)\), then the limit does not exist. Choose paths strategically: \(y=0\), \(x=0\), \(y=x\), \(y=mx\), \(y=x^2\), etc.
Theorem · Squeeze Theorem
If \(f(x,y) \le g(x,y) \le h(x,y)\) near \((a,b)\) and \(\lim f = \lim h = M\), then \(\lim g = M\).

Common use: If \(0 \le g(x,y) \le |x|\) or \(0 \le g(x,y) \le \sin^2 y\), then since \(|x|\to 0\) and \(\sin^2 y \to 0\), squeeze gives \(g \to 0\).
⚠ Important
L'Hôpital's Rule does NOT apply to multivariable limits. Use algebra, factoring, polar coordinates, or the squeeze theorem instead.
Definition · Continuity
\(f\) is continuous at \((a,b)\) iff: (1) \(f(a,b)\) is defined, (2) the limit exists, and (3) \(\lim_{(x,y)\to(a,b)} f(x,y) = f(a,b)\). Polynomials and rational functions (where denominator \(\ne 0\)) are continuous on their domains.
Example — Two-Path Test
Show \(\displaystyle\lim_{(x,y)\to(0,0)} \frac{x^2 - y^2}{x^2+y^2}\) does not exist.
Path 1: \(y=0\): \(\dfrac{x^2}{x^2} = 1\).
Path 2: \(x=0\): \(\dfrac{-y^2}{y^2} = -1\).
Different values \(\Rightarrow\) limit DNE.
§ 4.3

Partial Derivatives

Definition
\(f_x(x,y) = \dfrac{\partial f}{\partial x} = \displaystyle\lim_{h\to 0}\dfrac{f(x+h,y)-f(x,y)}{h}\)    (treat \(y\) as constant)

\(f_y(x,y) = \dfrac{\partial f}{\partial y}\)    (treat \(x\) as constant)
Theorem · Clairaut's Theorem
If \(f_{xy}\) and \(f_{yx}\) are both continuous on an open disk, then: \[f_{xy}(a,b) = f_{yx}(a,b)\] (Mixed partials are equal — order doesn't matter when both are continuous.)
Example
Let \(f(x,y) = x^2 - 3y^4\).
\(f_x = 2x\),   \(f_y = -12y^3\),   \(f_{xx} = 2\),   \(f_{yy} = -36y^2\),   \(f_{xy} = 0\)
§ 4.4

Tangent Planes and Linear Approximation

Tangent Plane to z = f(x,y) at (x₀, y₀, z₀)
\[ z = f(x_0, y_0) + f_x(x_0,y_0)(x-x_0) + f_y(x_0,y_0)(y-y_0) \]
Tangent Plane to F(x,y,z) = k (Level Surface) at (x₀,y₀,z₀)
\[ F_x(x_0,y_0,z_0)(x-x_0) + F_y(x_0,y_0,z_0)(y-y_0) + F_z(x_0,y_0,z_0)(z-z_0) = 0 \] The normal vector to this plane is \(\vec{n} = \nabla F(x_0, y_0, z_0)\).
Linearization (Linear Approximation)
\[ L(x,y) = f(x_0,y_0) + f_x(x_0,y_0)(x-x_0) + f_y(x_0,y_0)(y-y_0) \] Use: \(f(x,y) \approx L(x,y)\) for \((x,y)\) near \((x_0,y_0)\).
Normal Line (to level surface F(x,y,z)=k at P(x₀,y₀,z₀))
\[ \vec{r}(t) = \langle x_0 + F_x t,\; y_0 + F_y t,\; z_0 + F_z t \rangle \]
Example — from Quiz Week 7
Let \(f(x,y) = \sqrt{x^2+y^2}\). At \((3,4)\): \(f(3,4)=5\), \(f_x = \frac{x}{\sqrt{x^2+y^2}} = \frac{3}{5}\), \(f_y = \frac{4}{5}\).
Tangent plane: \(z = \frac{3}{5}(x-3) + \frac{4}{5}(y-4) + 5\)
Linearization: \(L(x,y) = \frac{3}{5}(x-3) + \frac{4}{5}(y-4) + 5\)
Approximation at (2.9, 4.1): \(L(2.9,4.1) = \frac{3}{5}(-0.1) + \frac{4}{5}(0.1) + 5 = -0.06+0.08+5 = 5.02\)
§ 4.5

Chain Rule

Chain Rule — One Intermediate Variable
If \(z = f(x,y)\), \(x=x(t)\), \(y=y(t)\), then: \[\frac{dz}{dt} = \frac{\partial f}{\partial x}\frac{dx}{dt} + \frac{\partial f}{\partial y}\frac{dy}{dt}\]
Chain Rule — Two Intermediate Variables (e.g., s and t)
If \(z = f(x,y)\), \(x=x(s,t)\), \(y=y(s,t)\): \[\frac{\partial z}{\partial s} = \frac{\partial f}{\partial x}\frac{\partial x}{\partial s} + \frac{\partial f}{\partial y}\frac{\partial y}{\partial s}, \qquad \frac{\partial z}{\partial t} = \frac{\partial f}{\partial x}\frac{\partial x}{\partial t} + \frac{\partial f}{\partial y}\frac{\partial y}{\partial t}\] Tip: Draw a tree diagram with \(f\) at top, \(x,y\) below, then the independent variables below those. Multiply along each branch, sum all paths to the variable you're differentiating with respect to.
Implicit Differentiation via Chain Rule
Given \(F(x,y)=0\) with \(y=f(x)\): \[\frac{dy}{dx} = -\frac{F_x}{F_y}\] Given \(F(x,y,z)=0\) with \(z=f(x,y)\): \[\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}, \qquad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z}\]
Example — Implicit Differentiation
Given \(e^z = xyz\), find \(\frac{\partial z}{\partial x}\).
Set \(F(x,y,z) = e^z - xyz\). \(F_x = -yz\), \(F_z = e^z - xy\).
\(\dfrac{\partial z}{\partial x} = -\dfrac{F_x}{F_z} = -\dfrac{-yz}{e^z-xy} = \dfrac{yz}{e^z-xy}\)
§ 4.6

Directional Derivatives and the Gradient

Definition · Directional Derivative
The directional derivative of \(f(x,y)\) in the direction of a unit vector \(\hat{u} = \langle a,b\rangle\): \[D_{\hat{u}}f(x,y) = \lim_{h\to 0} \frac{f(x+ah, y+bh) - f(x,y)}{h} = \nabla f(x,y)\cdot\hat{u}\] ⚠ Always normalize: \(\hat{u} = \frac{\vec{v}}{||\vec{v}||}\)
Gradient Vector
\[\nabla f(x,y) = \langle f_x(x,y),\; f_y(x,y)\rangle\] \[\nabla f(x,y,z) = \langle f_x,\; f_y,\; f_z\rangle\]
Theorem · Maximum Rate of Change
For differentiable \(f\) at \((x_0,y_0)\):
  • Maximum value of \(D_{\hat{u}}f(x_0,y_0)\) is \(||\nabla f(x_0,y_0)||\), achieved in direction \(\nabla f(x_0,y_0)\).
  • Minimum value is \(-||\nabla f(x_0,y_0)||\), in direction \(-\nabla f(x_0,y_0)\).
  • \(D_{\hat{u}}f = 0\) when \(\hat{u} \perp \nabla f\).
Example — from Quiz Week 8
Let \(f(x,y) = e^x\sin y\). Find \(D_{\hat{u}}f\) at \((1,\frac{\pi}{2})\) in direction \(\vec{v} = -\hat{i} = \langle -1,0\rangle\).
\(\nabla f = \langle e^x\sin y,\; e^x\cos y\rangle\). At \((1,\frac{\pi}{2})\): \(\nabla f = \langle e, 0\rangle\).
\(\hat{u} = \langle -1,0\rangle\) (already unit).
\(D_{\hat{u}}f = \langle e,0\rangle \cdot \langle -1,0\rangle = -e\).
Max directional derivative \(= ||\nabla f(1,\frac{\pi}{2})|| = e\), in direction \(\langle e, 0\rangle\) (or normalized \(\langle 1, 0\rangle\)).
§ 4.7

Maximum and Minimum Values

Critical Points
A point \((x_0,y_0)\) is a critical point of \(f\) if either:
  1. \(f_x(x_0,y_0) = 0\) AND \(f_y(x_0,y_0) = 0\), or
  2. One of the partial derivatives does not exist at \((x_0,y_0)\).
Theorem · Second Derivative Test
Let \(D(x,y) = f_{xx}f_{yy} - (f_{xy})^2\). At a critical point \((x_0,y_0)\):
  • If \(D > 0\) and \(f_{xx} > 0\): Local Minimum
  • If \(D > 0\) and \(f_{xx} < 0\): Local Maximum
  • If \(D < 0\): Saddle Point
  • If \(D = 0\): Inconclusive
Finding Absolute Extrema on a Closed Bounded Region D
  1. Find all critical points of \(f\) inside \(D\) and evaluate \(f\) at each.
  2. Find all extrema of \(f\) on the boundary of \(D\) (parametrize the boundary).
  3. The largest value from steps 1–2 is the absolute max; the smallest is the absolute min.
Example — from Notes Week 7
Find critical points of \(f(x,y) = y^3 + 3x^2y - 6x^2 - 6y^2 + 2\).
\(f_x = 6xy - 12x = 6x(y-2) = 0 \Rightarrow x=0\) or \(y=2\).
\(f_y = 3y^2 + 3x^2 - 12y = 0\).
If \(x=0\): \(3y(y-4)=0 \Rightarrow y=0\) or \(y=4\). Critical pts: \((0,0), (0,4)\).
If \(y=2\): \(3(4)+3x^2-24=0 \Rightarrow x^2=4 \Rightarrow x=\pm 2\). Critical pts: \((2,2),(-2,2)\).
\(D(0,0) = (6\cdot0-12)^2 - 0 = 144 > 0\), \(f_{xx}(0,0)=-12 < 0\) → Local Max.
\(D(0,4) = (6\cdot4-12)^2 = 144 > 0\), \(f_{xx}(0,4)=12 > 0\) → Local Min.
\(D(2,2)=(0)^2-(12)^2 = -144 < 0\) → Saddle point (same for \((-2,2)\)).
§ 5.1

Double Integrals over Rectangles

Definition
For a rectangle \(R = [a,b]\times[c,d]\), the double integral is the limit of the double Riemann sum: \[\iint_R f(x,y)\,dA = \lim_{\substack{m\to\infty\\n\to\infty}} \sum_{i=1}^m\sum_{j=1}^n f(x_i^*,y_j^*)\,\Delta x\,\Delta y\] Geometric meaning: if \(f(x,y)\ge 0\), this gives the volume of the solid above \(R\) and below \(z=f(x,y)\).
Theorem · Fubini's Theorem (Rectangle)
If \(f\) is continuous on \(R=[a,b]\times[c,d]\): \[\iint_R f(x,y)\,dA = \int_c^d\!\int_a^b f(x,y)\,dx\,dy = \int_a^b\!\int_c^d f(x,y)\,dy\,dx\] You can integrate in either order on a rectangle.
Theorem · Separable Functions
If \(f(x,y) = g(x)\cdot h(y)\) on \(R=[a,b]\times[c,d]\): \[\iint_R f(x,y)\,dA = \left(\int_a^b g(x)\,dx\right)\left(\int_c^d h(y)\,dy\right)\]
Average Value
\[f_{\text{avg}} = \frac{1}{\text{Area}(R)}\iint_R f(x,y)\,dA\]
§ 5.2

Double Integrals over General Regions

Type I Region
Bounded by \(x=a\), \(x=b\), \(y=g_1(x)\), \(y=g_2(x)\).
\[\iint_R f\,dA = \int_a^b\!\int_{g_1(x)}^{g_2(x)} f(x,y)\,dy\,dx\] (Outer integral: \(x\). Inner integral: \(y\) varies with \(x\).)
Type II Region
Bounded by \(y=c\), \(y=d\), \(x=h_1(y)\), \(x=h_2(y)\).
\[\iint_R f\,dA = \int_c^d\!\int_{h_1(y)}^{h_2(y)} f(x,y)\,dx\,dy\] (Outer integral: \(y\). Inner integral: \(x\) varies with \(y\).)
Reversing Order of Integration — The Strategy
  1. Sketch the region \(R\) described by the original limits.
  2. Describe the same region with the opposite variable as outer.
  3. Read off the new bounds from your sketch.
Example — Reversing Order (from Exam Review #14)
Reverse the order of \(\displaystyle\int_0^8\int_{\sqrt[3]{y}}^{2} e^{3x^2}\,dx\,dy\).

Step 1: Identify region: \(0 \le y \le 8\), \(\sqrt[3]{y}\le x\le 2\).
Step 2: Sketch — boundary curves are \(y=x^3\) and \(x=2\), with \(y=0\) at bottom.
Step 3: As Type I: \(x\) from 0 to 2, and for each \(x\), \(y\) from 0 to \(x^3\).
\[\int_0^2\int_0^{x^3} e^{3x^2}\,dy\,dx = \int_0^2 x^3 e^{3x^2}\,dx\] (Now integrable by substitution \(u=3x^2\)!)
Theorem · Volume between Two Surfaces
Volume of solid bounded above by \(z=f_2(x,y)\) and below by \(z=f_1(x,y)\) over region \(R\): \[V = \iint_R [f_2(x,y) - f_1(x,y)]\,dA\]
§ 5.3

Double Integrals in Polar Coordinates

The Core Conversion Formulas

Everything in polar coordinates builds from these four relationships:

CARTESIAN → POLAR
\[r^2 = x^2 + y^2\] \[\tan\theta = \dfrac{y}{x}\]
POLAR → CARTESIAN
\[x = r\cos\theta\] \[y = r\sin\theta\]
Area Element in Polar Coordinates
When converting \(dA\) to polar, you must include the Jacobian factor \(r\): \[dA = dx\,dy = r\,dr\,d\theta\] Never forget the extra \(r\)!
Double Integral in Polar — General Form
\[\iint_D f(x,y)\,dA = \int_\alpha^\beta\int_{r_1(\theta)}^{r_2(\theta)} f(r\cos\theta,\, r\sin\theta)\cdot r\,dr\,d\theta\] Substitute \(x = r\cos\theta\) and \(y = r\sin\theta\) into the integrand.
Common Polar Regions
Region DescriptionPolar Bounds
Full disk of radius \(a\)\(0\le r\le a\), \(0\le\theta\le 2\pi\)
Top half of disk of radius \(a\)\(0\le r\le a\), \(0\le\theta\le \pi\)
Right half of disk of radius \(a\)\(0\le r\le a\), \(-\frac{\pi}{2}\le\theta\le\frac{\pi}{2}\)
First quadrant\(r\ge 0\), \(0\le\theta\le\frac{\pi}{2}\)
Annulus: inner radius \(a\), outer radius \(b\)\(a\le r\le b\), \(0\le\theta\le 2\pi\)
Circle \(x^2+y^2=a^2\)\(r = a\)
Example — from Exam Review #17
Evaluate \(\displaystyle\iint_D (2x-y)\,dA\) where \(D\) is the top half of the disk of radius 5.

Convert: \(x=r\cos\theta\), \(y=r\sin\theta\), \(dA=r\,dr\,d\theta\).
Bounds: \(0\le r\le 5\), \(0\le\theta\le\pi\).
\[= \int_0^\pi\int_0^5 (2r\cos\theta - r\sin\theta)\,r\,dr\,d\theta = \int_0^\pi\int_0^5 (2r^2\cos\theta - r^2\sin\theta)\,dr\,d\theta\] \[= \int_0^\pi \left[\frac{2r^3}{3}\cos\theta - \frac{r^3}{3}\sin\theta\right]_0^5 d\theta = \int_0^\pi \left(\frac{250}{3}\cos\theta - \frac{125}{3}\sin\theta\right)d\theta\] \[= \frac{250}{3}[\sin\theta]_0^\pi - \frac{125}{3}[-\cos\theta]_0^\pi = \frac{250}{3}(0) + \frac{125}{3}(-1-1) = -\frac{250}{3}\]

Polar Coordinates — Complete Reference

When to Use Polar Coordinates
Switch to polar when the region or integrand involves:
  • Circles or sectors: \(x^2+y^2 = a^2\), \(x^2+y^2 \le a^2\)
  • Expressions like \(x^2+y^2\) in the integrand (becomes \(r^2\))
  • \(\sqrt{x^2+y^2}\) in the integrand (becomes \(r\))
  • Symmetric regions (full disk, half-disk, quarter-disk, annulus)
Converting the Region (Cartesian → Polar Bounds)
  1. Identify the shape of the region in \(xy\)-coordinates.
  2. Convert boundary curves using \(x=r\cos\theta\), \(y=r\sin\theta\), \(r^2=x^2+y^2\).
  3. Determine the range of \(\theta\) (which "wedge" of the plane?).
  4. Determine the range of \(r\) (inner/outer radius, possibly depending on \(\theta\)).
  5. The inner integral is always over \(r\); outer is over \(\theta\).
Worked Example — Region Conversion
Region: \(1 \le x^2+y^2 \le 4\) in the upper half-plane (\(y\ge 0\)).
Polar: \(r^2 = x^2+y^2\), so \(1\le r^2\le 4 \Rightarrow 1\le r\le 2\).
Upper half-plane: \(0\le\theta\le\pi\).
Bounds: \(\int_0^\pi\int_1^2 (\cdots) r\,dr\,d\theta\)
Worked Example — Converting Difficult Cartesian Integrals
Evaluate \(\displaystyle\int_0^3\int_0^{\sqrt{9-x^2}} e^{-(x^2+y^2)}\,dy\,dx\).

The region is the quarter-circle \(x^2+y^2\le 9\), \(x\ge 0\), \(y\ge 0\).
Polar bounds: \(0\le r\le 3\), \(0\le\theta\le\frac{\pi}{2}\).   Integrand: \(e^{-r^2}\).
\[= \int_0^{\pi/2}\int_0^3 e^{-r^2}\cdot r\,dr\,d\theta = \frac{\pi}{2}\int_0^3 re^{-r^2}\,dr = \frac{\pi}{2}\cdot\frac{1-e^{-9}}{2} = \frac{\pi(1-e^{-9})}{4}\]
★

Switching Order of Integration (Cartesian)

Step-by-Step Method
  1. Read the original bounds to find the region \(R\). E.g., \(\int_a^b\int_{g_1(x)}^{g_2(x)} f\,dy\,dx\) means \(a\le x\le b\), \(g_1(x)\le y\le g_2(x)\).
  2. Sketch the region in the \(xy\)-plane. Plot the boundary curves.
  3. Describe the region in the opposite order: find the overall \(y\)-range (outer), then for each fixed \(y\), find the \(x\)-range (inner).
  4. Write the new iterated integral.
Example — Exam Review #15
Reverse: \(\displaystyle\int_{-2}^{2}\int_{x^2}^{4} f(x,y)\,dy\,dx\).

Original region: \(-2\le x\le 2\), \(x^2\le y\le 4\).
Sketch: Region bounded above by \(y=4\), below by parabola \(y=x^2\) (or equivalently \(x=\pm\sqrt{y}\)).
Overall \(y\)-range: \(0\le y\le 4\).
For fixed \(y\): \(x\) ranges from \(-\sqrt{y}\) to \(\sqrt{y}\).
\[\int_0^4\int_{-\sqrt{y}}^{\sqrt{y}} f(x,y)\,dx\,dy\]
Study Tip
When you reverse order, the value of the integral does not change (by Fubini's theorem). The reason to reverse is usually to make integration possible — e.g., when \(e^{x^2}\), \(\frac{\sin x}{x}\), or \(\frac{x}{\ln x}\) appear as inner integrands and have no elementary antiderivative with respect to that variable.

Exam-Style Practice Problems

Based on the Test 2 Review Sheet · MAC 2313
PROBLEM 01 · §4.1 / §4.2

Find the domain and range of \(f(x,y) = \ln(2x - 3y + 1)\). For what set of points is \(f\) continuous?

Need \(2x-3y+1 > 0 \Rightarrow y < \frac{2x+1}{3}\).
Domain: \(\{(x,y)\mid 2x-3y+1 > 0\}\) — the open half-plane below the line \(y = \frac{2x+1}{3}\).
Range: \((-\infty, \infty) = \mathbb{R}\) (ln can take any real value when input is positive).
Continuity: \(f\) is continuous on its domain (same set), since \(\ln u\) is continuous for \(u>0\) and the argument is a continuous function.
PROBLEM 02 · §4.2 — Limits

Evaluate: \(\displaystyle\lim_{(x,y)\to(0,0)} \frac{2xy}{x^2+2y^2}\)

Try two paths:
Path 1 (\(y=0\)): \(\lim_{x\to 0}\frac{0}{x^2} = 0\).
Path 2 (\(x=y\)): \(\lim_{x\to 0}\frac{2x^2}{x^2+2x^2} = \lim_{x\to 0}\frac{2x^2}{3x^2} = \frac{2}{3}\).
Two different values \(\Rightarrow\) limit does not exist (DNE).
PROBLEM 03 · §4.3 + §4.6 — Partials & Gradient

Let \(f(x,y) = x^2 - 3y^4\).

  1. Find \(f_x\) and \(f_y\).
  2. Find the equation of the tangent plane at \((4, 1, 13)\).
  3. Find \(\nabla f(x,y)\).
  4. Find \(D_{\hat{u}}f(5,1)\) in the direction of \(\vec{v} = \langle 3,4\rangle\).
  5. Find the maximum value of \(D_{\hat{u}}f(5,1)\) and the direction it occurs.
(a) \(f_x = 2x\), \(f_y = -12y^3\).
(b) At \((4,1)\): \(f_x=8\), \(f_y=-12\). Tangent plane: \(z = 8(x-4) - 12(y-1) + 13 = 8x-12y-7\).
(c) \(\nabla f = \langle 2x, -12y^3\rangle\).
(d) At \((5,1)\): \(\nabla f = \langle 10, -12\rangle\). Normalize \(\vec{v}\): \(\hat{u} = \langle\frac{3}{5},\frac{4}{5}\rangle\). \(D_{\hat{u}}f = \langle 10,-12\rangle\cdot\langle\frac{3}{5},\frac{4}{5}\rangle = 6 - \frac{48}{5} = \frac{30-48}{5} = -\frac{18}{5}\).
(e) Max value \(= ||\nabla f(5,1)|| = \sqrt{100+144} = \sqrt{244} = 2\sqrt{61}\), in direction \(\langle 10,-12\rangle\) (or normalized).
PROBLEM 04 · §4.7 — Critical Points & 2nd Derivative Test

Find and classify all critical points of \(h(x,y) = \dfrac{x^4}{4} - xy + \dfrac{y^2}{2}\).

\(h_x = x^3 - y = 0 \Rightarrow y = x^3\).
\(h_y = -x + y = 0 \Rightarrow y = x\).
So \(x^3 = x \Rightarrow x(x^2-1)=0 \Rightarrow x = 0, 1, -1\).
Critical points: \((0,0)\), \((1,1)\), \((-1,-1)\).
\(h_{xx}=3x^2\), \(h_{yy}=1\), \(h_{xy}=-1\). \(D = 3x^2 - 1\).
At \((0,0)\): \(D = -1 < 0\) → Saddle point.
At \((1,1)\): \(D = 2 > 0\), \(h_{xx}=3>0\) → Local minimum.
At \((-1,-1)\): \(D = 2>0\), \(h_{xx}=3>0\) → Local minimum.
PROBLEM 05 · §5.2 — Double Integral with Sketching

Let \(R\) be bounded by \(y=\sqrt{x}\), \(y=1\), and \(x=0\). Evaluate \(\displaystyle\iint_R (y^2+x)\,dA\).

  1. Sketch and shade \(R\).
  2. Set up with order \(dy\,dx\).
  3. Set up with order \(dx\,dy\).
  4. Evaluate.
The region is bounded on the left by \(x=0\), above by \(y=1\), and below by \(y=\sqrt{x}\) (i.e., \(x=y^2\)).
As \(dx\,dy\) (Type II): \(\displaystyle\int_0^1\int_0^{y^2}(y^2+x)\,dx\,dy\)
\(= \int_0^1\left[y^2 x + \frac{x^2}{2}\right]_0^{y^2}dy = \int_0^1 \left(y^4 + \frac{y^4}{2}\right)dy = \int_0^1 \frac{3y^4}{2}\,dy = \frac{3}{2}\cdot\frac{1}{5} = \frac{3}{10}\).
PROBLEM 06 · §5.3 — Polar Coordinates

Evaluate \(\displaystyle\iint_D (2x-y)\,dA\) where \(D\) is the top half of the disk centered at the origin with radius 5.

Use polar: \(x=r\cos\theta\), \(y=r\sin\theta\), \(dA=r\,dr\,d\theta\). Bounds: \(0\le r\le 5\), \(0\le\theta\le\pi\).
\(= \int_0^\pi\int_0^5(2r\cos\theta - r\sin\theta)\cdot r\,dr\,d\theta = \int_0^\pi\int_0^5(2r^2\cos\theta - r^2\sin\theta)\,dr\,d\theta\)
\(= \int_0^\pi\left[\frac{2r^3}{3}\cos\theta - \frac{r^3}{3}\sin\theta\right]_0^5 d\theta = \int_0^\pi\left(\frac{250}{3}\cos\theta - \frac{125}{3}\sin\theta\right)d\theta\)
\(= \frac{250}{3}[\sin\theta]_0^\pi + \frac{125}{3}[\cos\theta]_0^\pi = 0 + \frac{125}{3}(-1-1) = \boxed{-\dfrac{250}{3}}\)
PROBLEM 07 · §5.2 — Reverse Order of Integration

Reverse the order of integration of \(\displaystyle\int_0^8\int_{\sqrt[3]{y}}^{2} e^{3x^2}\,dx\,dy\).

Original: \(0\le y\le 8\), \(y^{1/3}\le x\le 2\). Boundary: \(x=y^{1/3} \Rightarrow y=x^3\).
As Type I: \(0\le x\le 2\), \(0\le y\le x^3\).
\[\int_0^2\int_0^{x^3} e^{3x^2}\,dy\,dx = \int_0^2 x^3 e^{3x^2}\,dx\] (Let \(u=3x^2\), \(du=6x\,dx\): \(= \frac{1}{6}\int_0^{12}u\,e^u\,\frac{du}{2}\) … evaluate by parts.)
PROBLEM 08 · §4.5 — Chain Rule

Let \(f(x,y)\) be differentiable, \(x=x(s,t,u)\), \(y=y(s,t)\). Find \(\dfrac{\partial f}{\partial s}\) and \(\dfrac{\partial f}{\partial u}\).

\(\dfrac{\partial f}{\partial s} = \dfrac{\partial f}{\partial x}\dfrac{\partial x}{\partial s} + \dfrac{\partial f}{\partial y}\dfrac{\partial y}{\partial s}\)

\(\dfrac{\partial f}{\partial u} = \dfrac{\partial f}{\partial x}\dfrac{\partial x}{\partial u}\)   (since \(y\) does not depend on \(u\))
PROBLEM 09 · §2.6 — Quadric Surfaces

For each surface, (1) name it, (2) describe level curves or traces, (3) if a cylinder, identify the free variable and direction of rulings.

  1. \(z^2 - 2y^2 = 1\)
  2. \(z = x^2 + y^2\)
  3. \(z = x^2 - y^2\)
  4. \(z^2 = x^2 + y^2\)
(a) Missing \(x\): Hyperbolic cylinder, rulings ∥ \(x\)-axis. Traces in \(yz\): hyperbola \(z^2-2y^2=1\).
(b) Elliptic paraboloid (circular since coefficients equal). Level curves: \(x^2+y^2=k\), circles.
(c) Hyperbolic paraboloid (saddle). Level curves: \(x^2-y^2=k\), hyperbolas.
(d) Elliptic cone (circular). Level curves: \(x^2+y^2=k^2\), circles.
PROBLEM 10 · §4.7 — Absolute Extrema on Closed Region

Find the absolute maximum and minimum of \(f(x,y) = x^2+y^2-2y+1\) on the region \(R = \{(x,y)\mid x^2+y^2\le 4\}\).

Interior critical points: \(f_x=2x=0\), \(f_y=2y-2=0 \Rightarrow x=0, y=1\). Check: \((0,1)\in R\). \(f(0,1)=0\).
Boundary \(x^2+y^2=4\): parametrize \(x=2\cos\theta\), \(y=2\sin\theta\).
\(g(\theta) = 4\cos^2\theta+4\sin^2\theta-4\sin\theta+1 = 4-4\sin\theta+1 = 5-4\sin\theta\).
\(g'(\theta) = -4\cos\theta = 0 \Rightarrow \theta=\pi/2, 3\pi/2\).
\(g(\pi/2)=5-4=1\), \(g(3\pi/2)=5+4=9\). Endpoints: \(g(0)=g(2\pi)=5\).
Absolute max = 9 at \((0,-2)\). Absolute min = 0 at \((0,1)\).

Quick-Reference Formula Sheet

Gradient & Directional Derivative
\[\nabla f = \langle f_x, f_y\rangle \quad\text{(or }\langle f_x,f_y,f_z\rangle\text{)}\] \[D_{\hat{u}}f = \nabla f \cdot \hat{u}\quad(\hat{u}\text{ must be unit!})\] Max rate of change: \(||\nabla f||\), direction: \(\nabla f\)
Second Derivative Test
\[D = f_{xx}f_{yy} - (f_{xy})^2\] \(D>0, f_{xx}>0\): Local min
\(D>0, f_{xx}<0\): Local max
\(D<0\): Saddle  |  \(D=0\): Inconclusive
Tangent Plane (z = f(x,y))
\[z = f(x_0,y_0)+f_x(x-x_0)+f_y(y-y_0)\] Implicit differentiation: \(\dfrac{dy}{dx}=-\dfrac{F_x}{F_y}\)
Polar Coordinates
\[x=r\cos\theta,\; y=r\sin\theta,\; dA=r\,dr\,d\theta\] \[r^2=x^2+y^2,\;\tan\theta=y/x\]
Fubini's Theorem
Rectangle: \(\int_c^d\int_a^b f\,dx\,dy = \int_a^b\int_c^d f\,dy\,dx\)
Type I: \(\int_a^b\int_{g_1(x)}^{g_2(x)} f\,dy\,dx\)
Type II: \(\int_c^d\int_{h_1(y)}^{h_2(y)} f\,dx\,dy\)
Average Value
\[f_{\text{avg}} = \frac{\iint_R f(x,y)\,dA}{\text{Area}(R)}\] Volume between surfaces \(f_1,f_2\):
\[V = \iint_R [f_2-f_1]\,dA\]
⚠ Top Mistakes to Avoid on the Exam
  • Forgot to normalize when computing directional derivative — always make \(\hat{u}\) a unit vector first!
  • Forgot the \(r\) in \(dA = r\,dr\,d\theta\) for polar integrals.
  • Wrong D for SDT — it's \(f_{xx}f_{yy} - (f_{xy})^2\), NOT \(f_{xx}f_{yy} - f_{xy}^2\) without the square.
  • Reversing order without sketching — always draw the region first!
  • L'Hôpital on multivariable limits — it does NOT apply.
  • Wrong bounds for polar — full disk is \(0\le\theta\le 2\pi\), top half is \(0\le\theta\le\pi\).
  • Confused a cylinder with a quadric — if one variable is missing from the equation, it is a cylinder.
